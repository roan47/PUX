About

  • Viewport metatag
    This keeps the website from zooming out on small screens.
  • Widths as %
    This allows images to respond in size to the overall screen size.
  • Font size as em
    Set a px measurement on the body selector's font-size. All other font-sizes should be em to allow for ease in making adjustments to font sizes.
  • CSS Media Queries
    CSS instructions that wait for certain viewport widths and execute styles at those screen sizes.
  • JS Tweaks to UI
    There are times where you need some JavaScript to manage changes to the UI. (i.e. Hamburger Menu buttons, hiding certain elements, etc.)
