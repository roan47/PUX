What is JavaScript?

JavaScript is a language that can be interpreted by the browser and provides logic/interactivity to a website. While HTML provides the content/structure for the site and CSS provides formatting/style to a site, JavaScript adds things like form validation, sending data to other applications, changing styling when certain events occur, and more!

Info about JS

JavaScript's Role

JavaScript provides the interactivity/logic for a website. With JS, we can add or manipulate content on the site, change stylings, create animations, validate form inputs, and many other tasks that affect the User Experience.

JavaScript attaches to HTML much the same way CSS does. We will create a JS file and link it in the HTML via a script tag at the bottom of the body element.

<script src="/js/main.js"></>

Variables in JS

JS is not type-safe, so we are not able to protect variables from storing one type of data.
In JS, there are three types of variables:

  • var - a variable accessible globally throughout a JS file
  • let - limited scope variable only accessible in the scopes it was created
  • const - limited in scope and unable to be reassigned


Debugging in JS

Since JS isn't type-safe, there is a little bit more of a process in debugging JavaScript code. We must make full use of:

  • console.log() - sends messages to the browser's inspector
  • console.table() - organizes an array into a table
  • alert() - an unstylable popup
  • console debugger - browser's way to set breakpoints and test JS code
